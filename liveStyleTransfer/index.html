Live Style Transfer

  • Overview

    The project is an implementation of “Real-Time Style Transfer”. The Project covers using various technologies to extract “Style” data from a reference image and reconstruct the “input data” (a live video feed from a webcam) with the features mentioned. The model was trained with training using a dataset from COCO with 80,000+ images. The image on the right was used as our styling reference.
